# OceanLeo catalog rights notice

This file records the rights basis used by
`oceanleo-catalog/v1/manifest.json`. It is provenance evidence, not a new
general-purpose public license.

## OceanLeo first-party assets

Rows marked `LicenseRef-OceanLeo-First-Party` were already attested in the
production `platform_assets` inventory as owned or curated by OceanLeo with
commercial use and modification allowed. They may be copied into an OceanLeo
user's private fork, edited in the named OceanLeo editor and exported as part of
that user's work. No broader relicensing of the standalone source is asserted.

## External assets

External source terms remain controlling:

- Pixabay Content License: <https://pixabay.com/service/license/>
- Creative Commons Attribution-ShareAlike 3.0:
  <https://creativecommons.org/licenses/by-sa/3.0/>
- Creative Commons CC0 1.0:
  <https://creativecommons.org/publicdomain/zero/1.0/>

The manifest preserves provider, author, source page, license URL and required
attribution per item. A project with third-party dependencies preserves each
dependency's terms separately; the first-party project file does not erase or
replace them.
